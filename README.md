# OmniTools - 多功能工具集

## 📋 产品简介

OmniTools是一款多功能桌面应用，旨在为用户提供高效、便捷的工具集合，解决日常工作和生活中的各种问题。目前已集成「文件整理专家」和「空目录清理」两款实用工具，支持智能文件分类管理和目录清理，未来将持续扩展更多实用工具。

### 核心价值
- **跨平台**：支持Windows、Linux、macOS等多种操作系统
- **个性化**：支持多语言、主题切换等个性化设置
- **模块化**：动态加载工具插件，易于扩展

## 🚀 核心功能

### 1. 文件整理专家

**功能亮点**：
- **多维度分类**：支持按文件类型、时间、大小和自定义规则分类
- **可视化规则管理**：直观创建、编辑和管理整理规则
- **整理预览**：执行前预览分类结果，包含详细统计信息
- **批量操作**：一次性处理大量文件，提高效率
- **整理报告**：生成详细的整理报告，记录操作结果

**使用场景**：
- 整理下载文件夹中的杂乱文件
- 分类照片、文档、视频等不同类型的文件
- 按照时间顺序归档工作文件
- 根据自定义规则组织项目文件

### 2. 空目录清理

**功能亮点**：
- **智能扫描**：快速扫描指定目录中的空目录
- **树形视图**：以直观的树形结构展示目录层级
- **安全删除**：智能删除空目录，保留用户选择的根目录
- **统计分析**：提供详细的扫描统计信息

**使用场景**：
- 清理下载目录中产生的空文件夹
- 整理项目目录，移除无用的空目录
- 优化存储空间，删除无意义的空文件夹
- 保持文件系统结构整洁

### 3. 现代化用户界面

**设计特点**：
- **响应式布局**：适配不同屏幕尺寸
- **多标签页操作**：便捷切换不同功能模块
- **流畅动画**：平滑的过渡效果和交互反馈
- **蓝色主题**：专业、现代的视觉风格
- **直观导航**：侧边栏导航，快速访问各功能

### 4. 多语言支持

**语言选项**：
- 简体中文
- 英文

**切换方法**：在设置页面中选择所需语言，应用将实时更新所有界面文本。

### 5. 主题系统

**主题选项**：
- 蓝色主题（默认）
- 支持自定义主题

**设置方法**：在设置页面中选择或自定义主题，应用将立即应用新的视觉风格。

### 6. 自动更新系统

**功能特点**：
- **自动检查**：应用启动时自动检查新版本
- **智能通知**：发现新版本时显示更新提示
- **一键更新**：在应用内直接下载并安装更新
- **更新渠道**：支持稳定版和测试版两种更新渠道
- **安全可靠**：文件完整性校验，自动备份和回滚
- **更新历史**：查看历史版本和更新日志

**使用方法**：
1. **自动检查**：应用启动时自动检查更新（可在设置中关闭）
2. **手动检查**：在设置页面点击「检查更新」按钮
3. **查看详情**：点击更新通知中的「查看详情」查看完整更新日志
4. **立即更新**：点击「立即更新」按钮下载并安装新版本
5. **稍后提醒**：点击「稍后提醒」按钮，24小时后再次提醒

**更新设置**：
- **自动检查更新**：开启/关闭自动检查功能
- **更新渠道**：选择稳定版或测试版
- **显示更新通知**：开启/关闭更新通知
- **更新历史**：查看最近5个版本的更新记录

**安全保障**：
- 下载前验证文件完整性（SHA256校验）
- 更新前自动创建备份
- 安装失败时自动回滚到备份版本
- 用户数据和配置不受影响

### 7. 系统集成

**窗口管理**：
- 自动保存窗口位置和大小
- 下次启动时恢复上次状态

**通知系统**：
- 实时操作反馈
- 重要状态提醒
- 可自定义通知开关

**认证系统**：
- 支持用户登录/登出
- 安全的身份验证

## 📦 下载与安装

### 安装方式

 **下载预构建包**：
   访问 [GitHub Releases](https://github.com/cmdragon/OmniTools/releases) 下载对应平台的安装包

### 安装步骤

#### Windows
1. 下载 `OmniTools-windows.zip` 文件
2. 解压到任意位置
3. 运行 `OmniTools.exe` 启动应用

#### Linux
1. 下载 `OmniTools-linux.tar.gz` 文件
2. 解压到任意位置
3. 运行 `OmniTools` 可执行文件启动应用

#### macOS
1. 下载 `OmniTools-macos.tar.gz` 文件
2. 解压到任意位置
3. 运行 `OmniTools.app` 启动应用

## 🛠️ 使用指南

### 快速开始

1. **启动应用**：双击应用图标启动OmniTools
2. **选择工具**：在左侧边栏选择所需工具
3. **配置参数**：根据工具要求配置相关参数
4. **执行操作**：点击对应按钮执行操作
5. **查看结果**：查看操作结果和相关报告

### 文件整理专家使用指南

1. **选择文件夹**：点击「选择文件夹」按钮，选择要整理的文件夹
2. **配置规则**：（可选）在「规则管理」标签页配置整理规则
3. **预览结果**：点击「预览整理」按钮，查看整理效果
4. **执行整理**：点击「开始整理」按钮，执行文件整理
5. **查看报告**：在「整理报告」标签页查看整理结果

### 规则管理

**创建规则**：
1. 在「规则管理」标签页点击「添加规则」按钮
2. 选择规则类型（文件类型、时间、大小、自定义）
3. 配置规则参数
4. 设置目标文件夹
5. 点击「保存」按钮

**编辑规则**：
1. 在规则列表中选择要编辑的规则
2. 点击「编辑」按钮
3. 修改规则参数
4. 点击「保存」按钮

**删除规则**：
1. 在规则列表中选择要删除的规则
2. 点击「删除」按钮
3. 确认删除操作

### 空目录清理使用指南

1. **选择文件夹**：点击「选择文件夹」按钮，选择要扫描的目录
2. **开始扫描**：点击「扫描目录」按钮，开始扫描空目录
3. **查看结果**：在树形视图中查看扫描结果
4. **删除空目录**：点击「删除空目录」按钮，删除扫描到的空目录
5. **重新扫描**：删除后可重新扫描，确认清理效果

### 高级设置

**语言设置**：
- 点击顶部导航栏的「设置」按钮
- 在「语言」选项中选择所需语言
- 应用将自动更新界面语言

**主题设置**：
- 点击顶部导航栏的「设置」按钮
- 在「主题」选项中选择或自定义主题
- 应用将立即应用新主题

**通知设置**：
- 点击顶部导航栏的「设置」按钮
- 在「通知」选项中开启或关闭通知
- 保存设置

**更新设置**：
- 点击顶部导航栏的「设置」按钮
- 在「更新」选项中配置自动更新行为
- 选择更新渠道（稳定版/测试版）
- 开启或关闭自动检查更新
- 开启或关闭更新通知
- 点击「检查更新」手动检查新版本
- 点击「更新历史」查看历史版本

## ❓ 常见问题

### Q1: OmniTools支持哪些操作系统？
A: OmniTools目前支持Windows 10+、Linux Ubuntu 20.04+、macOS 12+等主流操作系统。

### Q2: 文件整理功能会影响文件的原始质量吗？
A: 不会，文件整理只是移动文件的位置，不会修改文件内容或质量。

### Q3: 如何添加自定义整理规则？
A: 在「文件整理专家」工具的「规则管理」标签页，点击「添加规则」按钮，选择「自定义规则」类型，然后配置规则参数。

### Q4: 整理大量文件时应用会卡顿吗？
A: 不会，应用采用异步处理和进度指示，即使处理大量文件也能保持界面响应流畅。

### Q5: 如何更新到最新版本？
A: OmniTools支持自动更新。应用启动时会自动检查新版本，发现更新时会显示通知。您也可以在「设置」页面点击「检查更新」按钮手动检查。点击「立即更新」即可下载并安装新版本。

### Q6: 更新会影响我的数据吗？
A: 不会。更新系统会在安装前自动创建备份，确保您的用户数据和配置不受影响。如果更新失败，系统会自动回滚到备份版本。

### Q7: 空目录清理会删除我选择的根目录吗？
A: 不会，空目录清理工具会智能识别并保留用户选择的根目录，只删除其内部的空子目录。

### Q8: 如何报告bug或请求新功能？
A: 您可以通过GitHub Issues（[https://github.com/cmdragon/OmniTools/issues](https://github.com/cmdragon/OmniTools/issues)）报告bug或提出新功能请求。
